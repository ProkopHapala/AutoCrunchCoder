# tests/Cpp_Train/CodeExamples

C++ header code examples used for model training, testing, and reference implementations.

## Files

Header-only libraries and sample implementations including:

- `AOIntegrals.h`, `Bspline.h`, `CubicBSpline.h`
- `CLCFGO.h`, `EwaldGrid.h`, `Grid2DAlgs.h/cpp`, `GridFF.h`
- `ForceField.h`, `Forces.h`, `InteractionsGauss.h`
- `InterpolateTricubic.h`, `InterpolateTrilinear.h`
- `Lingebra.h`, `Mat3.h`, `Multipoles.h`, `NBFF.h`
- `MMFFsp3_loc.h`, `UFF.h`, `eFF.h`, `RARFF_SR.h`
- `PotentialFlow.h`, `TerrainHydraulics.h/cpp`
- `Vec2.h`, `Vec3.h`, `quaternion.h`

These are intended as reference/training material and may not all build standalone.
