# pyCruncher2

Reorganized scientific-computing module for AutoCrunchCoder. Contains the `scientific/` sub-package with CAS bindings, GPU helpers, element data, and plotting utilities. These are cleaner, package-structured versions of the corresponding `pyCruncher/` modules.

## Subdirectory

- `scientific/` — CAS (Maxima), GPU (OpenCL/CUDA), periodic table data, and shared plotting helpers.
