# examples

Curated usage examples demonstrating the main AutoCrunchCoder capabilities. Each subdirectory covers one work-stream.

## Subdirectories

- `knowledge/` — Paper pipeline examples: PDF extraction, summarization, BibTeX classification, Chroma vector store ingestion, and RAG queries (DeepSeek + Gemini). Demonstrates the offline-first PDF→SQLite pipeline.
- `MCP/` — Model Context Protocol servers (chemistry, LAMMPS, Maxima) and matching clients. Shows how an LLM consumes external tools via MCP.
- `scientific/` — Scientific computing examples: Maxima derivative computation, OpenCL/CUDA N-body simulation, Slater-orbital integration. End-to-end from symbolic math to GPU kernels.

Run each example from the repository root after activating the project environment.
